## Scope

## The three roles (inherited)

1. **Interpret** — unstructured nursing data → structured observations. Free-text notes, CNA reports, ambient signals → typed schemas.
2. **Mediate** — when RL importance weights and channel survival probabilities diverge, reason about WHY. Context: trajectory type, medication effects, role-specific knowledge, atypical presentations.
3. **Explain** — technical mediation output → structured handoff content that survives the channel.

## Pattern

1. **Read rl-specialist** — what does the RL model say matters?
2. **Read simulation** — what ambient signals exist in the patient state?
3. **Read clinical** — what nursing context drives the mediation reasoning?
4. **Design or refine** the mediation protocol, structured JSON arbitration schema, reasoning field requirements.

## Guardrails

- Structured output ALWAYS. No free-text LLM output in the pipeline.
- Reasoning fields are the audit trail, not decoration.
- The mediation is not a black box — every arbitration decision must be traceable.
- The LLM doesn't write the handoff. It shapes how the RL agent prioritizes information for transmission.
